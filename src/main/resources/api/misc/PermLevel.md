# PermLevel
### Informacja
PermLevel to określenie permisji użytkownika

| Poziom uprawnień | Nazwa         |
|------------------|---------------|
| 0                | MEMBER        |
| 1                | STAZYSTA      |
| 2                | CHATMOD       |
| 3                | HELPER        |
| 4                | MODERATOR     |
| 5                | ADMINISTRATOR |
| 10               | DEVELOPER     |


### Przykłady
```json
{
  "permLevel": "DEVELOPER"
}
```
