# Member/Info

## Path
`api/member/{member}/info`

## Opis
Zwraca informacje na temat danego użytkownika

## Autoryzacja
Poprzez IP

## Query
| Key    | Value          |
|--------|----------------|
| member | ID użytkownika |

## Przykłady
```json
{
  "success": true,
  "data": {
    "Patrz Userinfo.md"
  }
}
```
